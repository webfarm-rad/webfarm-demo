# Garage Express - Автосервис в Нижнем Новгороде

Демо-сайт для автосервиса Garage Express.

## Информация

- **Название:** Garage Express
- **Город:** Нижний Новгород
- **Адрес:** ГСК Лада, 214
- **Телефон:** +7 (930) 290-88-22
- **Yandex Maps:** https://yandex.com/maps/org/garageexpress/4711189230/

## Технологии

- HTML5
- CSS3 (Custom Properties, Grid, Flexbox)
- Vanilla JavaScript (ES6+)
- Font Awesome 6
- Google Fonts (Oswald + Roboto)

## Дизайн

- **Цветовая схема:** Red (#D32F2F) + Dark (#1A1A1A) + Orange (#FF6F00)
- **Шрифты:** Oswald (заголовки) + Roboto (текст)
- **Стиль:** Industrial, professional, automotive

## Структура

- **index.html** - Главная страница с полным функционалом
- **css/style.css** - 1600+ строк стилей
- **js/main.js** - 280+ строк JavaScript
- **robots.txt** - Файл для поисковых роботов
- **sitemap.xml** - Карта сайта

## Особенности

- Hero slider с 3 слайдами
- Табы с услугами (6 категорий)
- Marquee с логотипами брендов
- Анимированные счётчики статистики
- Карусель отзывов
- FAQ аккордеон
- Форма обратного звонка
- Scroll reveal анимации
- Адаптивный дизайн
- WhatsApp интеграция
