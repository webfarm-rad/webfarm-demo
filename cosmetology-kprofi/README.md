## 🚀 Деплой на Netlify

### Подготовка

```bash
cd /Users/radium/agent\ webfarm/sites/cosmetology-kprofi
```

### Через Netlify CLI

```bash
# Установить Netlify CLI (если не установлен)
npm install -g netlify-cli

# Авторизоваться
netlify login

# Инициализировать проект
netlify init

# Задеплоить
netlify deploy --prod
```

### Через Netlify UI

1. Зайти на netlify.com
2. Выбрать "Add new site" → "Deploy manually"
3. Перетащить папку `cosmetology-kprofi`
4. Дождаться деплоя
5. Настроить кастомный домен (если нужно)

## 🌐 Настройка домена

После покупки клиентом домена:

1. В Netlify: Site settings → Domain management → Add custom domain
2. Добавить домен клиента
3. Настроить DNS записи (Netlify предоставит инструкции)
4. Дождаться SSL сертификата (автоматически через Let's Encrypt)
