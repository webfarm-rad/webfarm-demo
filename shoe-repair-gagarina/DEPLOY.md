# Инструкция по деплою

## 🚀 Быстрый старт на Netlify

### Шаг 1: Подготовка
1. Убедитесь, что все файлы на месте:
   - index.html, about.html, services.html, works.html, contacts.html
   - css/style.css
   - js/script.js
   - robots.txt, sitemap.xml, .htaccess

### Шаг 2: Деплой на Netlify
1. Зайдите на https://www.netlify.com/
2. Войдите или зарегистрируйтесь
3. Нажмите "Add new site" → "Deploy manually"
4. Перетащите папку `shoe-repair-gagarina` целиком
5. Дождитесь завершения деплоя

### Шаг 3: Настройка домена
1. В настройках сайта → "Domain management"
2. "Add custom domain"
3. Введите домен клиента (например: shoe-repair-gagarina.ru)
4. Следуйте инструкциям по настройке DNS

### Шаг 4: SSL
SSL сертификат включится автоматически через несколько минут после настройки домена.

## 🌐 Альтернативные варианты хостинга

### GitHub Pages (бесплатно)
1. Создать репозиторий на GitHub
2. Загрузить файлы
3. Settings → Pages → Source: main branch
4. Сайт будет доступен на `username.github.io/repo-name`

### Vercel (бесплатно)
1. Зарегистрироваться на https://vercel.com/
2. Импортировать из Git или загрузить напрямую
3. Настроить домен

### Обычный shared hosting
1. Загрузить файлы через FTP в папку `public_html`
2. .htaccess уже настроен
3. SSL настроить через панель хостинга

## 🆘 Решение проблем

### Сайт не открывается после деплоя
- Проверьте, что все файлы загружены
- Проверьте консоль браузера на ошибки (F12)
- Убедитесь, что index.html в корневой папке

### Мобильная версия не работает
- Очистите кеш браузера
- Проверьте, что viewport meta-тег присутствует
- Проверьте в режиме инкогнито

### Карта не отображается
- Проверьте интернет-соединение
- Убедитесь, что iframe не заблокирован
- Попробуйте в другом браузере

### Формы не отправляются
- Проверьте номер WhatsApp (без пробелов: 79272073428)
- Попробуйте кликнуть на ссылку WhatsApp напрямую
- Проверьте консоль на JavaScript ошибки
